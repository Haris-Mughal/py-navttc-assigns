## Week 03: Debugging Practice
**Objective:** Identifying and resolving the three primary types of programming errors.
- **Syntax Errors:** Fixing structural mistakes (e.g., missing colons).
- **Runtime Errors:** Implementing `try-except` blocks for division by zero.
- **Semantic Errors:** Correcting logical flaws in mathematical formulas.

## Week 04: Functions & Logic
**Objective:** Modularizing code for reusability and clarity.
- Implemented a `fahrenheit_to_kelvin()` function.
- Focused on **Parameter Passing** and **Return Values**.
- Included input validation to ensure numerical integrity.

## Week 05: Data Structures
**Objective:** Efficient data management using Python Collections.
- **Lists:** Practical use of `.append()`, `.sort()`, and `.reverse()`.
- **Dictionaries:** Managing key-value pairs using `.update()`, `.get()`, and `.pop()`.
- Demonstrates how to organize complex data points effectively.

## Week 06: File Handling
**Objective:** Creating persistent applications through File I/O.
- Demonstrated the **CRUD** cycle: Create, Read, Update, Delete.
- Implemented the `with` statement (Context Manager) for secure file closing.
- Handled file existence checks using the `os` module.
